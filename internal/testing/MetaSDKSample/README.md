# MetaSDKSample

Used for developing and regression testing SDK features.

From this directory run:
`./generate-projects.sh`

Open the generated Xcode project and start coding!

## Signing

If you need to build to a device, you will need to download the Provisioning Profile:
MetaSDK Sample Local Development - com.meta.MetaSDKSample.localDevelopment

You can find it on
https://www.internalfb.com/intern/apple/download-provisioning-profile/ under the section Meta Platforms, Inc.
